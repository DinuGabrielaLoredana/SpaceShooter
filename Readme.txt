1. Controls
-----------

Key Controls :

    Escape         - Exit
    S              - Save Game
    L              - Load Game


    W    - Move Player1 Forward
    A    - Move Player1 Backward
    S    - Strafe Player1 Left
    D    - Strafe Player1 Right
    Z    - Player1 Fire
    X    - Player1 Rotate
    Q    - Player1 Explosion
   
    
    Cursor Up      - Move Player2 Forward
    Cursor Down    - Move Player2 Backward
    Cursor Left    - Strafe Player2 Left
    Cursor Right   - Strafe Player2 Right
    Space          - Player2 Fire
    Return         - Player2 explosion
    R              - Player2 Rotate
    
Mouse Controls :
    
    Left Button    - Use Mouse Look
